# Crowd Meter Live - Core Features

## What The App Does

Crowd Meter Live is a sports-focused mobile app that measures ambient noise levels in real time and turns matchday energy into shareable fan moments.

It uses the device microphone to estimate decibel intensity, visualizes the result with animated UI, and provides viral social hooks for sports communities.

## Core Features

- Real-time noise measurement using the microphone.
- Live dB estimation and instant loudness labeling (for example: Quiet, Moderate, Loud, Very Loud).
- Animated circular ring and dynamic bars that react to current sound intensity.
- Camera-backed measurement mode to create an immersive live-stadium overlay while recording.
- One-tap start/stop measurement flow with permission handling for microphone and camera.

## Screen Breakdown

### Live Meter

- Primary measurement screen.
- Shows current dB value in real time.
- Displays motion-based visual feedback for crowd intensity.
- Supports live camera background during active measurement.

### Fan Challenge

- Presents a fan-focused scorecard concept based on crowd noise performance.
- Encourages viral sharing with challenge-oriented copy.
- Built for engagement through social competition (`#CrowdRoarChallenge`).

### Hype Feed

- Highlights trending fan momentum concepts (rank + rivalry prompts).
- Includes shareable rivalry callouts to drive social interaction.
- Supports fan-versus-fan style engagement loops (`#SectionBattle`).

## Design Direction

- White-first visual standard with translucent glass-style surfaces.
- Sporty, high-energy accents and bold typography.
- Consistent tab-based navigation across all screens.

## Typical User Flow

1. Open Live Meter.
2. Grant microphone (and optionally camera) permissions.
3. Start measurement and view live dB feedback.
4. Open Fan Challenge or Hype Feed.
5. Share score or rivalry challenge to social platforms.
